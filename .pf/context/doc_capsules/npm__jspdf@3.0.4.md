# jspdf@3.0.4 (npm)

## Quick Start

```sh
npm install jspdf --save
# or
yarn add jspdf
```

## Examples

```javascript
import { jsPDF } from "jspdf";

// Default export is a4 paper, portrait, using millimeters for units
const doc = new jsPDF();

doc.text("Hello world!", 10, 10);
doc.save("a4.pdf");
```
